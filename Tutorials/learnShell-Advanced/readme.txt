A collection of scripts that go along with the learnshell.org bash scripting course.
This directory contains the second half of the tutorial's scripts, titled "Advanced Tutorials".
The scripts from the first half, "Learn the Basics", are found in the "learnShell-Basics" directory.

Lessons in Tutorial:
-Special Variables
-Bash trap command
-File Testing
-Input Parameter Parsing (Lesson does not exist yet on website)
-Pipelines 
-Process Substitution
-Regular Expressions (Lesson does not exist yet on website)
-Special Commands sed,awk,grep,sort (Lesson does not exist yet on website)
