A collection of scripts that go along with the learnshell.org bash scripting course.
This directory contains the first half of the tutorial's scripts, titled "Learn the Basics".
The scripts from the second half, "Advanced Tutorials", are found in the "learnShell-Advanced" directory.

Lessons in the Tutorial:
-Hello, World!
-Variables
-Passing Arguments to the Script
-Arrays
-Array-Comparison
-Basic Operators
-Basic String Operations
-Decision Making
-Loops
-Shell Functions
